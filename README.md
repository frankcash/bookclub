# bookclub

Website for Hacker Book Club

## Installation

Download from [Github](https://github.com/frankcash/bookclub).

Then run:

```
$ lein deps
```
Make sure you have [Postgresql](http://www.postgresql.org/).
## Running

```
$ lein ring server-headless
```
